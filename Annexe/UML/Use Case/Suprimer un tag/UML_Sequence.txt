@startuml
title Supprimer un tag - Diagramme de séquence

participant User as U
participant System as S
database Database as D

ref over U,S,D : Cas d’usage "Consulter les recommandations" ou "Rechercher par filtre ou nom"

U -> S : Choisis le tag à suprimer
S -> D : Transmet la demande

alt Tag n'existe pas ou le tag n'est pas présent sur le film

  D --> S : N'effectue pas la demande
  S --> U : Notifie que le tag n'existe pas

else

  D --> S : Suprime le tag
  S --> U : Notifie la supression du tag

end


@enduml
